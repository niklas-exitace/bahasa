## Prepositions

| Bahasa | English | Notes |
|--------|---------|-------|
| dari | from | |
| di | in, at, on | static location |
| ke | to (a place) | movement/direction |
| sama | with/same | |
| untuk | for | |
| tanpa | without | |
| di sini | here | |

**Key distinction:**
- **di** = at/in (location, static) - "Saya tinggal **di** Bali"
- **ke** = to (direction, movement) - "Saya pergi **ke** Bali"
- **dari** = from (origin) - "Saya **dari** Bali"

## Quick Tips

1. **Three types of "good"**: bagus (things), baik (people), enak (food)
2. **"Selamat"** is used as a greeting prefix: Selamat pagi = Good morning
3. **Location words**: di (at/in), ke (to), dari (from)
4. **"Sekali"** comes AFTER adjectives: mahal sekali = very expensive
5. **Sudah/Belum** pair: sudah (already) vs belum (not yet)
6. **Sebelum/Sesudah** pair: sebelum (before) vs sesudah (after)
